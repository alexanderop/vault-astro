## Source Summary

Anthropic frames Project Glasswing as a defensive response to a new cyber reality: frontier models can now find and chain software vulnerabilities at a level close to top human experts, so the only viable move is to get those capabilities into defensive workflows before attackers normalize them.

The initiative brings together major infrastructure and security companies, extends access to dozens of additional critical-software organizations, and commits substantial credits and donations to help secure both proprietary and open-source systems.

## Key Claims

- Claude Mythos Preview has already found thousands of high-severity vulnerabilities, including issues in major operating systems and browsers.
- The cost and expertise required to find and exploit vulnerabilities is dropping fast because frontier models reason about code and exploits well.
- Defensive deployment cannot wait for perfect governance; Anthropic's answer is a coordinated industry program with limited access, disclosure discipline, and shared learning.
- Open-source maintainers are a strategic bottleneck, so Anthropic is pairing model access with direct funding for security organizations and foundations.

## Concrete Details

- Launch partners include AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, the Linux Foundation, Microsoft, NVIDIA, and Palo Alto Networks.
- Anthropic says it is committing up to `$100M` in usage credits and `$4M` in direct donations tied to open-source security work.
- Anthropic plans to report publicly within 90 days on lessons learned, vulnerabilities fixed, and best practices that can be disclosed.
